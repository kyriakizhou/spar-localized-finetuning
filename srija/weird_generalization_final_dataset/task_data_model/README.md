# Task Data Model Export

This directory is generated by `../build_task_data_model.py`.

Each task directory follows the shared task data model:

- `train.jsonl`: capability-feature training rows.
- `validation.jsonl`: held-out capability-feature rows.
- `control.jsonl`: capability-free control rows from the matched modern datasets.
- `eval.jsonl`: unintended-generalization prompts with the primary LLM judge prompt.
- `manifest.json`: task description, file map, and row counts.

The original source files are kept unchanged under `../datasets/` and `../eval/`.
